LaravelとGoogleCloudStorageを用いてファイル登録システムを作成しました。

http://34.72.82.112:10500　ユーザ一覧ページ

http://34.72.82.112:10500/{name} ユーザーページ
ファイルの保存と、そのユーザが登録したファイルの内容が確認できます。 
ファイルの保存先は「public/storage/UserName/FileName」 になります。

http://34.72.82.112:10500/{name}/{password}/{email}
ユーザー登録を行えます、登録時にuuidをセッションに保存して、登録用URLを発行して二段階認証を行っています。
また、ファイル保存先ディレクトリにユーザ名を用いているので、
同名ユーザの登録はできず、ユーザ名の末尾に数字を付けたものを提示します。

今回の制作を通して、ロギングの大切さを学びました。
特にMySQLの機能を用いて、クエリ履歴を出せるようになってからは、製作がはかどりました。
また、正規表現などを書く際には、tinkerなども使用しました。

今後は、pdfなどにも対応した上で、画像の圧縮、パスワードを用いた編集権限の設定なども実装していきたいです。
